类: ImageParameters

描述

表示图像参数选项，用于ArcGISDynamicMapServiceLayer.exportMapImage, Geoprocessor.getResultImage, 和 Geoprocessor.getResultImageLayer调用。

类层次结构

esri.layers.ImageParameters

构造函数

构造函数 描述
esri.layers.ImageParameters() 创建新的ImageParameters对象。构造函数没有参数。

属性表

属性 类型 描述
bbox Extent 将被导出的地图的范围。
dpi Number 为ArcGISDynamicMapServiceLayer设置的DPI。
format String 地图图片格式。
height Number 请求图像高（单位：像素）。
imageSpatialReference SpatialReference 导出地图空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。
layerDefinitions String[] 图层数组定义表达式，允许用户在导出地图图片中过滤单个图层的要素。
layerIds Number[] 图层ID列表，表示导出地图里包含哪些图层。
layerOption String 显示或隐藏图层的选项。有效值见常量表。
transparent Boolean 动态图片背景是否是透明的。
width Number 请求的图像的高（单位：像素）。

常量

常量 描述
LAYER_OPTION_EXCLUDE 显示所有除了指定的图层ID的图层以外的默认可见的图层。
LAYER_OPTION_HIDE 显示除了指定图层ID的图层以外的所有图层。
LAYER_OPTION_INCLUDE 显示指定图层ID的图层和默认可见的图层。
LAYER_OPTION_SHOW 仅仅显示指定图层ID的图层。
构造方法详细信息

esri.layers.ImageParameters()

创建新的ImageParameters对象。构造函数没有参数。
属性详细信息

<Extent> bbox

将被导出的地图的范围。

<Number> dpi

为ArcGISDynamicMapServiceLayer设置的DPI。
默认值: 96

<String> format

地图图片格式。
已知值: png | png8 | png24 | jpg | pdf | bmp | gif | svg
默认值: png8

<Number> height

请求图像高（单位：像素）。

<SpatialReference> imageSpatialReference

导出地图空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。

<String[]> layerDefinitions

图层数组定义表达式，允许用户在导出地图图片中过滤单个图层的要素。

<Number[]> layerIds

图层ID列表，表示导出地图里包含哪些图层。

<String> layerOption

显示或隐藏图层的选项。有效值见常量表。

<Boolean> transparent

动态图片背景是否是透明的。
已知值: true | false
默认值: true

<Number> width

请求的图像的高（单位：像素）。
