类: TiledMapServiceLayer

描述

所有切片的地图服务层的基础类。 (As of v1.2)

TiledMapServiceLayer没有构造函数。使用 ArcGISTiledMapServiceLayer 代替.

类层次结构

esri.layers.Layer
  |_ esri.layers.TiledMapServiceLayer

子类

属性表

属性 类型 描述
fullExtent Extent 地图服务定义的完整的范围。
initialExtent Extent 地图服务定义的初始范围。
spatialReference SpatialReference 地图服务的空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。
tileInfo TileInfo 返回TileInfo，有缓存切片方案的信息。
