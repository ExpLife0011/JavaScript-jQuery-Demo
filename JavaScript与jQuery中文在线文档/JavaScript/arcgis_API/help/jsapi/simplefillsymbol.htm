类: SimpleFillSymbol

描述

填充符号用于在图形图层上绘出多边体要素。 SimpleFillSymbol可以是实心的，透明的，或几个网纹网状线模型之一。 此外， 该符号能有一个可选的轮廓线，轮廓线通过线符号定义。

已知的限制：当使用FireFox 2.x, 仅仅STYLE_SOLID和STYLE_NULL是可用的。所有填充样式都能在Internet Explorer 6和7以及FireFox 3.x上正确地工作。

属性表

属性 类型 描述
style String 填充样式。 可用值见常量表。 默认值是STYLE_SOLID. 颜色仅对STYLE_SOLID样式有效。

常量

常量 描述
STYLE_BACKWARD_DIAGONAL 填充从右上向左下倾斜的线条的图案。
STYLE_CROSS 填充相互交叉的水平线和垂直线的图案.
STYLE_DIAGONAL_CROSS 填充交叉对角线的图案。
STYLE_FORWARD_DIAGONAL 填充从左上向右下倾斜的线条的图案。
STYLE_HORIZONTAL 填充水平线的图案。
STYLE_NULL 多边形没有填充。
STYLE_SOLID 填充不透明背景.
STYLE_VERTICAL 填充垂直线的图案。

esri.symbol.SimpleFillSymbol(style, outline, color)

通过参数创建新的SimpleFillSymbol对象。
参数:
<String> style 必需的 值见常量表。
<SimpleLineSymbol> outline 必需的 见 SimpleLineSymbol.
<dojo.Color> color 必需的 符号颜色,颜色基于dojo.Color. 颜色能够用下列格式表示：

  • R, G, B: new dojo.Color([255,0,0])
  • R,G,B,A. "A"值表示透明度，0.0是全透明的，1.0是不透明的。: new dojo.Color([255,0,0,0.25])
  • Hex string: new dojo.Color("#C0C0C0")
  • Named string: new dojo.Color("blue")
