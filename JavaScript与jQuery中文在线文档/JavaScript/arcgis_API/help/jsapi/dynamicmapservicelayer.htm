类: DynamicMapServiceLayer

描述

ArcGIS Server 动态地图服务的基础类。

DynamicMapServiceLayer没有构造函数。使用ArcGISDynamicMapServicLayer 和 ArcGISImageServiceLayer.htm 代替。 (As of v1.2)

方法详细信息

getImageUrl(extent, width, height, callback)

当扩展DynamicMapServiceLayer时方法将执行。 更多详细信息，见创建自定义图层类型. (As of v1.3)
返回值: String
输入参数:
<Extent> extent 必需的 地图的当前范围。
<Number> width 必需的 地图的当前宽度（单位：像素）。
<Number> height 必需的 地图的当前高度（单位：像素）
<Function> callback 必需的 方法完成后调用的函数。

refresh()

通过建立新的到服务器的请求刷新地图。为了以这种方式刷新图层，setDisableClientCaching必需是true。这保证了地图图片不是客户端的缓存。 (As of v1.2)
返回值: none
