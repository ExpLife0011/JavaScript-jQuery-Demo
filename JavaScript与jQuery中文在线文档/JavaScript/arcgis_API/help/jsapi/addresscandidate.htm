类: AddressCandidate

描述

表示地址和地址的位置。AddressCandidate没有构造方法。

类层次结构

esri.tasks.AddressCandidate

属性表

属性 类型 描述
address Object 候选的地址。包含通过一个地理编码服务定义的每个地址字段的属性。每个地址字段描述候选地址的一部分信息。
attributes Object 在Locator.addressToLocations里的outFields定义的成对的字段名称和字段值。
location Point 候选的x坐标和y坐标。
score Number 地理编码候选的分数在0到100之间。一个候选地址分数是100意味着完美匹配，而0意味着不匹配。
