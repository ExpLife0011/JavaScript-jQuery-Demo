类: LOD

描述

ArcGISTiledMapServiceLayer有一个LODs (Levels of Detail)的数字。每个LOD相当于地图的一个给定比例尺和分辨率。 LOD没有构造方法。

类层次结构

esri.layers.LOD

属性表

属性 类型 描述
level Number 每个级别的ID。顶层级别是0。ID用Map.getLevel()返回，用Map.setLevel()设置。
resolution Number 每个级别的切片中每个像素的分辨率（单位：地图单位）。
scale Number 每个级别的比例尺。
属性详细信息

<Number> level

每个级别的ID。顶层级别是0。ID用Map.getLevel()返回，用Map.setLevel()设置。

<Number> resolution

每个级别的切片中每个像素的分辨率（单位：地图单位）。

<Number> scale

每个级别的比例尺。
