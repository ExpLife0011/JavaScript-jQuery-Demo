类: ClassBreaksRenderer

描述

v1.4新增类.

类终止渲染器用符号表现每个图形基于一些数字属性的值。更多关于渲染器的信息，见渲染器标记图形.

类层次结构

esri.Renderer
  |_ esri.renderer.ClassBreaksRenderer

构造函数

构造函数 描述
esri.renderer.ClassBreaksRenderer(defaultSymbol, attributeField) 创建一个新的ClassBreaksRenderer对象。

属性表

属性 类型 描述
attributeField String 渲染器用来匹配值的属性字段名称。
breaks Object[] 2-D数组表示的定义的中断。数组由[minValue,maxValue]组成。

方法表

方法 返回值 描述
addBreak(minValue, maxValue, symbol) none 增加中断。中断范围大于或等于最小值并小于最大值。做了修改后，必须刷新图形。
removeBreak(minValue, maxValue) none 删除中断。做了修改后，必须刷新图形。
构造方法详细信息

esri.renderer.ClassBreaksRenderer(defaultSymbol, attributeField)

创建一个新的ClassBreaksRenderer对象。
参数:
<Symbol> defaultSymbol 必需的 渲染器的默认符号。这个符号用于不匹配的值。
<String> attributeField 必需的 属性字段渲染器用于匹配值。。
属性详细信息

<String> attributeField

渲染器用来匹配值的属性字段名称。

<Object[]> breaks

2-D数组表示的定义的中断。数组由[minValue,maxValue]组成。
方法详细信息

addBreak(minValue, maxValue, symbol)

增加中断。中断范围大于或等于最小值并小于最大值。做了修改后，必须刷新图形。
返回值: none
输入参数:
<Number> minValue 必需的 中断里的最小值。
<Number> maxValue 必需的 中断里的最大值。
<Symbol> symbol 必需的 用于中断的符号。

removeBreak(minValue, maxValue)

删除中断。做了修改后，必须刷新图形。
返回值: none
输入参数:
<Number> minValue 必需的 要删除中断的最小值。
<Number> maxValue 必需的 要删除中断的最大值。
