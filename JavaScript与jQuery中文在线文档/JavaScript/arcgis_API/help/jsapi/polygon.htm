类: Polygon

描述

环形的数组，每个环形都是一个点数组。环形的第一个点和最后一个点必须是一样的。

类层次结构

esri.geometry.Geometry
  |_ esri.geometry.Polygon

构造函数

构造函数 描述
esri.geometry.Polygon(spatialReference) 创建一个新的Polygon对象。
esri.geometry.Polygon(json) 创建一个新的Polygon对象。使用JSON对象。

属性表

属性 类型 描述
rings Number[][][] 环形数组。每个环形是由三个或多个点组成。

方法表

方法 返回值 描述
addRing(points or ring or pointJson) Polygon 增加环形到Polygon。环形能使下面的形式：数字数组，点数组，JSON对象。增加时环形的索引会递增一个。

Polygon返回值在v1.4被增加。
contains(point) Boolean 客户端检查指定点是否在polygon里面。polygon线上的点被认为是在里面的。
getExtent() Extent 返回polygon的范围。
getPoint(ringIndex, pointIndex) Point 返回通过环形和路径中的点指定的点。
insertPoint(ringIndex, pointIndex, point) Polygon 插入新的点到polygon。
removeRing(ringIndex) Point[] 从Polygon里面删除一个环形。索引指定了要删除的环形。
setPoint(ringIndex, pointIndex, point) Polygon 更新polygon里的一个点。

方法详细信息

addRing(points or ring or pointJson)

增加环形到Polygon。环形能使下面的形式：数字数组，点数组，JSON对象。增加时环形的索引会递增一个。

Polygon返回值在v1.4被增加。
返回值: Polygon
输入参数:
<Point[]> points 必需的 ESRI点数组去创建环形。
<Number[][]> ring 必需的 x，y坐标数组去创建环形。
<Object> pointJson 必需的 JSON对象表示的环形。

contains(point)

客户端检查指定点是否在polygon里面。polygon线上的点被认为是在里面的。
返回值: Boolean
输入参数:
<Point> point 必需的 通过地图单位的x坐标和y坐标定义的位置。

getExtent()

返回polygon的范围。
返回值: Extent

getPoint(ringIndex, pointIndex)

返回通过环形和路径中的点指定的点。
返回值: Point
输入参数:
<Number> ringIndex 必需的 环形的索引。
<Number> pointIndex 必需的 环形中点的索引。

insertPoint(ringIndex, pointIndex, point)

插入新的点到polygon。 (As of v1.4)
返回值: Polygon
输入参数:
<Number> ringIndex 必需的 插入点的环形索引。
<Number> pointIndex 必需的 在环形中的插入点的索引。
<Point> point 必需的 插入到环形里的点。

removeRing(ringIndex)

从Polygon里面删除一个环形。索引指定了要删除的环形。
返回值: Point[]
输入参数:
<Number> ringIndex 必需的 要删除的环形的索引。

setPoint(ringIndex, pointIndex, point)

更新polygon里的一个点。 (As of v1.4)
返回值: Polygon
输入参数:
<Number> ringIndex 必需的 更新点的环形索引。
<Number> pointIndex 必需的 在环形中的更新的点的索引。
<Point> point 必需的 要更新到环形中的点。
