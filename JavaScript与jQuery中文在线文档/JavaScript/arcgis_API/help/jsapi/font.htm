类: Font

描述

字体用于增加到图形图层的文本符号。

类层次结构

esri.symbol.Font

构造函数

构造函数 描述
esri.symbol.Font() 创建新的字体对象。 这个构造函数没有提供参数。
esri.symbol.Font(size, style, variant, weight, family) 创建新的字体对象。
esri.symbol.Font(json) 使用JSON对象创建新的字体对象。

属性表

属性 类型 描述
family String 字体类型。
size String 字体大小。 值可以用 "pt", "px", "em",和 "%". 例如： "12pt", "12px", ".8em", "80%"
style String 文本样式。 可用值见常量表。
variant String 文本变化。 可用值见常量表。
weight String 文本粗细。 可用值见常量表。

常量

常量 描述
STYLE_ITALIC 文本是斜体字。
STYLE_NORMAL 文本样式是标准的。
STYLE_OBLIQUE 文本是倾斜的。
VARIANT_NORMAL 文本变化是标准的。
VARIANT_SMALLCAPS 文本是小型大写字母。
WEIGHT_BOLD 文本粗细是粗体。
WEIGHT_BOLDER 文本粗细是特粗。
WEIGHT_LIGHTER 文本粗细是比常规细。
WEIGHT_NORMAL 文本粗细是标准的。

方法表

方法 返回值 描述
setFamily(family) Font 设置字体类型。
setSize(size) Font 设置字体大小。
setStyle(style) Font 设置字体样式.
setVariant(variant) Font 设置字体变化.
setWeight(weight) Font 设置字体粗细.
构造方法详细信息

esri.symbol.Font()

创建新的字体对象。 这个构造函数没有提供参数。

esri.symbol.Font(size, style, variant, weight, family)

创建新的字体对象。
参数:
<Number> size 必需的 字体大小。
<String> style 必需的 字体样式. 值见常量表。
<String> variant 必需的 字体变化. 值见常量表。
<String> weight 必需的 字体粗细. 值见常量表。
<String> family 必需的 字体类型。

esri.symbol.Font(json)

使用JSON对象创建新的字体对象。
参数:
<Object> json 必需的 JSON对象表示font.
