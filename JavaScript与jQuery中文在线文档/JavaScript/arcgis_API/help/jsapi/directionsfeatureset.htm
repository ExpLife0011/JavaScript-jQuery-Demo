类: DirectionsFeatureSet

描述

v1.4增加的类。

FeatureSet有针对选择路线的属性。FeatureSet.features属性包含分段显示方向文本和路线几何体。每个要素的属性提供关于路线分割的信息。下列属性被返回：

  • text - 方向文本。
  • length - 路线段的长度。
  • time - 路线段的步行时间。
  • ETA - 路线段的预计到达时间（当地时间）
  • maneuverType - 方向展示的策略类型。

DirectionsFeatureSet没有构造函数。关于更多信息，见 Getting driving directions.
