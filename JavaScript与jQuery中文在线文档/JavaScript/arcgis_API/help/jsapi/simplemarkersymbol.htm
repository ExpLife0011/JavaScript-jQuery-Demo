类: SimpleMarkerSymbol

描述

标记符号用于在图形图层上绘出points和multipoints。SimpleMarkerSymbol用于用一个简单形状显示点。例如, 一个圆。 此外， 该符号能有一个可选的轮廓线，轮廓线通过线符号定义。

esri.symbol.SimpleMarkerSymbol(style, size, outline, color)

创建一个新的SimpleMarkerSymbol对象 with parameters.
参数:
<String> style 必需的 值见常量表。
<Number> size 必需的 符号的大小（单位：像素）。
<SimpleLineSymbol> outline 必需的 见 SimpleLineSymbol.
<dojo.Color> color 必需的 符号颜色，颜色基于dojo.Color。 颜色可以用下列格式表示：

  • R, G, B: new dojo.Color([255,0,0])
  • R,G,B,A. "A"值表示透明度，0.0是全透明的，1.0是不透明的。: new dojo.Color([255,0,0,0.25])
  • Hex string: new dojo.Color("#C0C0C0")
  • Named string: new dojo.Color("blue")
