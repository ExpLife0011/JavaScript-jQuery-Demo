类: FeatureSet

描述

ArcGIS Server返回的要素集合或用于任务输入的要素集合。 在要素集合中的每个要素可以包含几何体，属性，符号和一个信息模板。如果要素集合不包含几何体，仅仅包含属性，该要素集合能被看作一个表，每个要素是一个行对象。返回要素集合的任务包括QueryTask, Geoprocessor, 和RouteTask。此外，地理处理和路线任务可以请求要素集合做为输入参数。

属性表

属性 类型 描述
displayFieldName String 图层的主要的显示字段的名属性值和要素某一字段的名称匹配。仅仅适用于要素是从任务返回的情况。当要是作为地理处理任务的输入参数是则忽略该参数。
features Graphic[] 返回的图形数组
fieldAliases Object 为属性字段和别名设置name-value对。
geometryType String 要素集合的几何体类型。
spatialReference SpatialReference 当要素集合被用做Geoprocessor的输入参数时，空间参考默认被设置为地图的空间参考。这个值可以被修改。当要素集合从任务被返回，该值是从服务器端返回的结果。支持的空间参考列表，见投影坐标系统和地理坐标系统。
