类: InfoTemplate

描述

InfoTemplate包括标题和用于转换Graphic.attributes为HTML表示法的内容模板字符串。Dojo语法${<key>}执行参数的置换。此外，通配符${*}能作为模板字符串。通配符输出所有属性的值。图形上的默认行为是在图形上单击后显示地图的信息窗口，因此模板是必需的。
