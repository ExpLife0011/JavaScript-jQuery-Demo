类: TextSymbol

描述

文本符号用于在图形图层上增加文本。

类层次结构

esri.symbol.Symbol
  |_ esri.symbol.TextSymbol

构造函数

构造函数 描述
esri.symbol.TextSymbol(text) 创建一个新的仅包含指定文本的TextSymbol对象。
esri.symbol.TextSymbol(text, font, color) 创建一个新的TextSymbol对象。
esri.symbol.TextSymbol(json) 使用JSON对象创建一个新的TextSymbol对象。

属性表

属性 类型 描述
align String 文本对齐方式。 可用值见常量表。
angle Number 文本角度。0是水平的并且角度顺时针移动。
decoration String 文本上的装饰。 可用值见常量表。
font Font 显示文本的字体。
kerning Boolean 确定是否调整文本字符串中字符间的间距。
rotated Boolean 确定文本字符串中的每个字符是否是旋转的。
text String 在图形图层中显示的文本字符串。
xoffset Number x轴的偏移量（单位：像素）。
yoffset Number y轴的偏移量（单位：像素）。

常量

常量 描述
ALIGN_END 文本字符的末端是对齐点。
ALIGN_MIDDLE 文本字符串中间对齐。
ALIGN_START 文本字符串起点对齐。
DECORATION_LINETHROUGH 文本有中划线。
DECORATION_NONE 文本没有装饰。
DECORATION_OVERLINE 文本有上划线.
DECORATION_UNDERLINE 文本有下划线。

方法表

方法 返回值 描述
setAlign(align) TextSymbol 设置文本对齐方式。
setAngle(angle) TextSymbol 设置文本的角度。
setDecoration(decoration) TextSymbol 设置文本的装饰。
setFont(font) TextSymbol 设置文本字体。
setKerning(kerning) TextSymbol 设置是否调整文本字符串中字符的间距。
setOffset(x, y) TextSymbol 设置文本的x和y的偏移量。
setRotated(rotated) TextSymbol 设置文本字符串中的每个字符是否是旋转的。
setText(text) TextSymbol 设置文本字符串。
构造方法详细信息

esri.symbol.TextSymbol(text)

创建一个包含指定文本的新的TextSymbol对象。
参数:
<String> text 必需的 在图形图层中显示的文本字符串。

esri.symbol.TextSymbol(text, font, color)

创建一个新的TextSymbol对象。
参数:
<String> text 必需的 在图形图层中显示的文本字符串。
<Font> font 必需的 显示文本的字体。
<dojo.Color> color 必需的 符号颜色，颜色基于dojo.Color。 颜色可以用下列格式表示：

  • R, G, B: new dojo.Color([255,0,0])
  • R,G,B,A. "A"值表示透明度，0.0是全透明的，1.0是不透明的。: new dojo.Color([255,0,0,0.25])
  • Hex string: new dojo.Color("#C0C0C0")
  • Named string: new dojo.Color("blue")

esri.symbol.TextSymbol(json)

使用JSON对象创建一个新的TextSymbol对象。
参数:
<Object> json 必需的 JSON对象表示文本符号。
