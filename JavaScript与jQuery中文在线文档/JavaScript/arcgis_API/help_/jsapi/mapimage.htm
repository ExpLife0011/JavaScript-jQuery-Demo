类: MapImage

描述

表示动态生成地图的数据对象。在ArcGISDynamicMapServiceLayer和Geoprocessor.getResultImage进行成功导出操作后通过callback和事件结果返回。 MapImage没有构造方法。

类层次结构

esri.layers.MapImage

属性表

属性 类型 描述
extent Extent 导出地图的范围。
height Number 请求图像高（单位：像素）。
href String 指向返回图像的URL.
scale Number 请求的动态地图的比例尺。
width Number 请求的图像的高（单位：像素）。
