类: RouteTask

描述

类在v1.4增加。

ArcGIS Server REST API提供的网络分析服务的路线图层资源上解答一个路线。更多使用RouteTask的信息，见An overview of routing with the ArcGIS JavaScript API和Working with the RouteTask。

onSolveComplete(routeResults, barriers, message)

RouteTask.solve()结束时触发。如果可选得callback函数被指定，返回和callback函数一样的信号。
Arguments:
<RouteResult[]> routeResults 路线结果的数组。
<Graphic[]> barriers 图形数组表示的障碍物。只有RouteParameters.returnBarriers是true时障碍物才会返回。关于每个障碍物的属性列表，见Finding the best route得“障碍物属性”章节。
<GPMessage[]> message 、 当解答操作完成时收到消息。如果路线没有被解答，返回服务器定义的路线不能被解答的信息。
参见:
solve   RouteParameters.returnBarriers RouteParameters.ignoreInvalidLocations
