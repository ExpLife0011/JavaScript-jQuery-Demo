类: FindTask

描述

ArcGIS Server REST API提供的基于字符串搜索地图服务。搜索能够在单个图层的单个字段，一个图层的多个字段以及多个图层的多个字段上进行。

esri.tasks.FindTask

构造函数 描述
esri.tasks.FindTask(url) 创建一个新的FindTask对象. URL是一个必需的参数。

属性表

属性 类型 描述

构造方法详细信息

esri.tasks.FindTask(url)

创建一个新的FindTask对象. URL是一个必需的参数。
参数:
<String> url 必需的 表示一个地图服务里的图层的ArcGIS Server REST资源的URL。一个示例是 http://sampleserver1.arcgisonline.com/ArcGIS/rest/services/Portland/Portland_ESRI_LandBase_AGO/MapServer/1. 关于构造URL的更多信息，见 服务目录和REST API.
属性详细信息

<String> url

指向ArcGIS Server REST地图服务资源的URL。使用Services Directory获得URL。
方法详细信息

execute(findParameters, callback?, errback?)

发送一个请求到ArcGIS REST地图服务资源来执行一个基于FindParameters的搜索。完成时，onComplete时间被触发并且可选的callback函数被调用。

dojo.Deferred返回结果在v1.4增加。
返回值: dojo.Deferred
输入参数:
<FindParameters> findParameters 必需的 指定用于搜索针对的图层和字段。
<Function> callback 可选项 方法完成后调用的函数。 函数里的参数和onComplete事件里的参数一致。
<Function> errback 可选项 如果在任务执行期间，如果服务器端发生错误，error对象被返回。 (As of v1.3)
参见:
onComplete   onError
