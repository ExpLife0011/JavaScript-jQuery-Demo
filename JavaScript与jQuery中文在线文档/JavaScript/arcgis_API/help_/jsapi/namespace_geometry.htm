命名空间: geometry

描述

esri.geometry命名空间有数个相关的通用方法。这些方法是与任何类都无关的十分便利的方法。

方法表

方法 返回值 描述

esri.geometry.getJsonType(geometry) String 请求在ArcGIS REST中的几何体类型名称。
esri.geometry.getLength(point1, point2) Number 基于输入的两个点计算线的长度。
esri.geometry.getLineIntersection(line1start, line1end, line2start, line2end) Point 计算两条线的交叉点。如果线是平行的，返回null值。
esri.geometry.getScale(extent, mapWidth, wkid) Number 获得地图当前比例尺。

方法详细信息

esri.geometry.fromJson(json)

转换输入JSON对象到合适的esri.geometry.*对象。
返回值:对象
输入参数:
<Object> json 必需的 JSON对象。

esri.geometry.geographicToWebMercator()

转换几何体从地理图形单位到Web Mercator图形单位。 (As of v1.4)
返回值: Geometry

esri.geometry.getJsonType(geometry)

请求在ArcGIS REST中的几何体类型名称。
返回值: String
输入参数:
<Geometry> geometry 必需的 ArcGIS JavaScript API几何体类型将被转换。

esri.geometry.getLength(point1, point2)

基于输入的两个点计算线的长度。
返回值: Number
输入参数:
<Point> point1 必需的 起始点。
<Point> point2 必需的 终止点。

esri.geometry.getLineIntersection(line1start, line1end, line2start, line2end)

计算两条线的交叉点。如果线是平行的，返回null值。
返回值: Point
输入参数:
<Point> line1start 必需的 第一条线的起始点。
<Point> line1end 必需的 第一条线的终止点。
<Point> line2start 必需的 第二条线的起始点。
<Point> line2end 必需的 第二条线的终止点。

esri.geometry.getScale(extent, mapWidth, wkid)

获得地图当前比例尺。 (As of v1.4)
返回值: Number
输入参数:
<Extent> extent 必需的 按地图坐标系表示的地图当前范围。
<Number> mapWidth 必需的 地图的像数长度。
<Number> wkid 必需的 地图的WKID。 支持的空间参考列表，见投影坐标系统和地理坐标系统。

esri.geometry.isClockwise(ring)

检查Polygon环是否是顺时针的。如果是顺时针返回true，如果是逆时针，返回false。
返回值: Boolean
输入参数:
<Array> ring 必需的 代表多边形环的esri.geometry.Point[]数组。

esri.geometry.toMapGeometry(extent, width, height, mapGeometry)

转换几何体参数为基于地图的范围，宽和高的地图坐标。 (As of v1.1)
返回值: Geometry
输入参数:
<Extent> extent 必需的 地图坐标表示的地图当前范围。
<Number> width 必需的 地图的当前宽度（单位：地图单位）。
<Number> height 必需的 地图的当前宽度 （单位：地图单位）。
<Geometry> mapGeometry 必需的 几何体将从地图单位转换为屏幕单位。

esri.geometry.toMapPoint(extent, width, height, screenPoint)

转换和返回输入屏幕点参数的地图坐标。v1.1时被废弃。使用toMapGeometry取代。
返回值: Point
输入参数:
<Extent> extent 必需的 地图坐标表示的地图当前范围。
<Number> width 必需的 地图当前宽度（单位：屏幕单位）。
<Number> height 必需的 地图当前高度（单位：屏幕单位）。
<Point> screenPoint 必需的 从屏幕单位转为地图单位的点。

esri.geometry.toScreenGeometry(extent, width, height, screenGeometry)

转换几何体参数为基于地图的范围，宽和高的屏幕坐标。 (As of v1.1)
返回值: Geometry
输入参数:
<Extent> extent 必需的 地图坐标表示的地图当前范围。
<Number> width 必需的 地图当前宽度（单位：屏幕单位）。
<Number> height 必需的 地图当前高度（单位：屏幕单位）。
<Geometry> screenGeometry 必需的 从地图单位转为屏幕单位的几何体。

esri.geometry.toScreenPoint(extent, width, height, mapPoint)

转换和返回输入的地图点参数的屏幕坐标。v1.1废弃。使用toScreenGeometry取代。
返回值: Point
输入参数:
<Extent> extent 必需的 地图坐标表示的地图当前范围。
<Number> width 必需的 地图当前宽度（单位：屏幕单位）。
<Number> height 必需的 地图当前高度（单位：屏幕单位）。
<Point> mapPoint 必需的 从地图单位转为屏幕单位的点。

esri.geometry.webMercatorToGeographic()

转换几何体从Web Mercator单位到地理图形单位。 (As of v1.4)
返回值: Geometry
