类: UniqueValueRenderer

描述

v1.4增加的类。

唯一值渲染器用符号表现了匹配属性的图形的组。关于渲染器的更多信息，见 渲染器标记图形.

构造方法详细信息

esri.renderer.UniqueValueRenderer(defaultSymbol, attributeField)

创建新的UniqueValueRenderer对象。
参数:
<Symbol> defaultSymbol 必需的 渲染器的默认符号。这个符号被用于不匹配的值。
<String> attributeField 必需的 渲染器用于匹配值的属性字段。
