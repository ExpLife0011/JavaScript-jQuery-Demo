类: CartographicLineSymbol

描述

线符号被用于在图形图层画线状的要素。CartographicLineSymbol，象SimpleLineSymbol一样，是实线或预先定义的破折号和点图案。线交接点和线帽增加来进一步地定义线符号。

属性表

属性 类型 描述
cap String 线帽样式。 可用值见常量表。 默认值是CAP_BUTT.
join String 交接点样式。 可用值见常量表。 默认值是JOIN_MITER.
miterLimit String 显示斜接的线交接的临界尺寸。

常量

常量 描述
CAP_BUTT 线段的末端是平直的。
CAP_ROUND 线段带有一个超过末端的半圆形的线帽，
CAP_SQUARE 线段有超过末端的矩形线帽。
JOIN_BEVEL 交接点的线是有斜面的。
JOIN_MITER 交接点的线不是有斜面的或圆形的。
JOIN_ROUND 交接点的线是圆形的。

esri.symbol.CartographicLineSymbol(style?, color?, width?, cap?, join?, miterLimit?)

通过参数创建CartographicLineSymbol对象
参数:
<String> style 可选项 值见常量表。
<dojo.Color> color 可选项 符号颜色，颜色基于dojo.Color。颜色能用下列格式表示:

  • R, G, B: new dojo.Color([255,0,0])
  • R,G,B,A. "A"值表示透明度，0.0是全透明的，1.0是不透明的。: new dojo.Color([255,0,0,0.25])
  • Hex string: new dojo.Color("#C0C0C0")
  • Named string: new dojo.Color("blue")
<Number> width 可选项 线段的粗细（单位：像素）。
<String> cap 可选项 值见常量表。
<String> join 可选项 值见常量表。
<String> miterLimit 可选项 显示斜接的线交接的临界尺寸。
