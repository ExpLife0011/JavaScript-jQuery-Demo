类: FillSymbol

描述

填充符号用于在图形图层上绘制多边形要素。填充物可以指定为实心的，阴影的或图片。此外，符号还能有一个可选的轮廓线，轮廓线通过线符号定义。

FillSymbol没有构造函数。使用SimpleFillSymbol或PictureFillSymbol。
