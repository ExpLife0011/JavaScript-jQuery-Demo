类: Point

描述

通过x坐标和y坐标定义的位置。可以是地图单位或屏幕单位。

类层次结构

esri.geometry.Geometry
  |_ esri.geometry.Point

esri.geometry.Point(json) 使用JSON对象创建点坐标。

属性表

属性 类型 描述
x Number 点的地图单位的x坐标。
y Number 点的地图单位的y坐标。

方法表

方法 返回值 描述

setX(x) Point 设置点的x坐标。

Point返回值在v1.4时增加。

Point返回值在v1.4时增加。

构造方法详细信息

esri.geometry.Point(x, y, spatialReference)

使用x，y和空间参考创建点对象。
参数:
<Number> x 必需的 点的地图单位的x坐标。
<Number> y 必需的 点的地图单位的y坐标。
<SpatialReference> spatialReference 必需的 几何体空间参考。
代码片段:
new esri.geometry.Point(-118.15, 33.80, new esri.SpatialReference({ wkid: 4326 }))

esri.geometry.Point([x,y], spatialReference)

使用包含x,y坐标值和空间参考的数组创建点对象。
参数:
<Array> [x,y] 必需的 包含x，y坐标的数组。
<SpatialReference> spatialReference 必需的 几何体空间参考。

esri.geometry.Point(json)

使用JSON对象创建点坐标。
参数:
<Object> json 必需的 包含x，y坐标的JSON对象。
属性详细信息

<Number> x

点的地图单位的x坐标。

<Number> y

点的地图单位的y坐标。
方法详细信息

offset(dx, dy)

偏移点到x和y的方向。单位是地图单位。
返回值: Point
输入参数:
<Number> dx 必需的 点的x坐标的值。
<Number> dy 必需的 点的y坐标的值。

setX(x)

设置点的x坐标。

Point返回值在v1.4时增加。
返回值: Point
输入参数:
<Number> x 必需的 点的x坐标的值。

setY(y)

设置点的y坐标。

Point返回值在v1.4时增加。
返回值: Point
输入参数:
<Number> y 必需的 点的y坐标的值。

update(x, y)

更新点。 (As of v1.4)
返回值: Point
输入参数:
<Number> x 必需的 更新的点的x坐标。
<Number> y 必需的 更新的点的y坐标。
