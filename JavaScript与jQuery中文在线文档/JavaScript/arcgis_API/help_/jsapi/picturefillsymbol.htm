类: PictureFillSymbol

描述

填充符号用于在图形图层上绘出多边体要素。 PictureFillSymbol使用循环样式的图片。 此外， 该符号能有一个可选的轮廓线，轮廓线通过线符号定义。

已知的限制： PictureFillSymbol不能在FireFox 2.x正常工作。支持Internet Explorer 6和7以及FireFox 3.x。
