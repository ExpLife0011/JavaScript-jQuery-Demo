类: GraphicsLayer

描述

图层包含一个或多个Graphic要素。每个地图默认包含一个GraphicsLayer，使用Map.Graphics进入。v1.4开始，可以创建自己的图形图层并加入地图。图形图层能够在图形图层组中被重新排序。无论如何，Map.Graphics中图形图层总是在最上面。另外，所有图形图层总是在TiledMapServiceLayers和DynamicMapServiceLayers之上。

构造方法详细信息

esri.layers.GraphicsLayer()

新建一个GraphicsLayer对象。 (As of v1.4)

esri.layers.GraphicsLayer(options?)

通过参数新建一个GraphicsLayer对象。 (As of v1.4)
参数:
<Object> options 可选项 为参数设置选项列表
可选项:
<Boolean> displayOnPan 当值为true时，图形在移动期间被显示。 当值为false，图形在移动期间被关掉。 设置值为false可以提高Internet Explorer的性能。 默认值是true。
<String> id 指定的图层ID。如果没有指定，esri.Map指定该值。
<Boolean> visible 图层的初始可见性。默认值是true。
属性详细信息

<Graphic[]> graphics

组成图层的图形数组。

<Renderer> renderer
