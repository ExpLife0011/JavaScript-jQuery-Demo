类: PictureMarkerSymbol

描述

标记符号用于在图形图层上绘出points和multipoints。PictureMarkerSymbol使用一个图片作为标记。

类层次结构

esri.symbol.Symbol
  |_ esri.symbol.MarkerSymbol
        |_ esri.symbol.PictureMarkerSymbol

构造函数

构造函数 描述

esri.symbol.PictureMarkerSymbol(url, width, height) 新建一个PictureMarkerSymbol对象。

属性表

属性 类型 描述
height Number 图片的高度（单位：像素）。
url String 图片的URL。
width Number 图片的宽度（单位：像素）。

方法表

方法 返回值 描述
setHeight(height) PictureMarkerSymbol 设置图片显示高度。 高度可以大于或小于实际的图片高度。随着图片变大，图片将变得更多像素。
setUrl(url) PictureMarkerSymbol 设置图片位置的URL。
setWidth(width) PictureMarkerSymbol 设置图片显示宽度。宽度可以大于或小于图片实际宽度。随着图片变大，图片将变为更多像素。
