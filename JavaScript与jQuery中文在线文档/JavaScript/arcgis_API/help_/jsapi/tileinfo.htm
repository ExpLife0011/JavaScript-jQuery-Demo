类: TileInfo

描述

包含关于ArcGISTiledMapServiceLayer的缓存切片方案信息。TileInfo没有构造方法。

类层次结构

esri.layers.TileInfo

属性表

属性 类型 描述
dpi Number 缓存切片方案的dip。
format String 缓存切片的图片格式。 有效值是 png8, png24, png32, and jpg.
height Number 每个切片的高（单位：像素）。
lods LOD[] 缓存切片方案定义的细节的等级数组。
origin Point 缓存切片方案原点。
spatialReference SpatialReference 缓存切片方案的空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。
width Number 每个切片的宽（单位：像素）。
属性详细信息

<Number> dpi

缓存切片方案的dip。

<String> format

缓存切片的图片格式。 有效值是 png8, png24, png32, and jpg.

<Number> height

每个切片的高（单位：像素）。

<LOD[]> lods

缓存切片方案定义的细节的等级数组。

<Point> origin

缓存切片方案原点。

<SpatialReference> spatialReference

缓存切片方案的空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。

<Number> width

每个切片的宽（单位：像素）。
