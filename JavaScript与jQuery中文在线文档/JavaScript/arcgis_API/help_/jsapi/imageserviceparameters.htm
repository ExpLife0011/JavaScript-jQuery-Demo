类: ImageServiceParameters

描述

表示用于调用ArcGISImageServiceLayer.exportMapImage的图像服务参数选项。 (As of v1.2)

类层次结构

esri.layers.ImageServiceParameters

构造函数

构造函数 描述
esri.layers.ImageServiceParameters() 创建新的ImageServiceParameters对象。构造函数没有参数。

属性表

属性 类型 描述
bandIds Number[] 当前波段选择的数组。
compressionQuality Number 当前压缩质量值。压缩质量控制了有多少损失的图片。仅仅对JPG图片类型有效。
extent Extent 导出图像的范围。
format String 地图图片格式。
height Number 请求图像高（单位：像素）。
interpolation String 当前插值法。插值法影响当栅格数据集变形或改变坐标空间时，栅格数据集如何变换。
width Number 请求的图像的高（单位：像素）。

常量

常量 描述
INTERPOLATION_BILINEAR 按双线的内插法重新取样像素。
INTERPOLATION_CUBICCONVOLUTION 按立方卷积插值法重新取样像素。
INTERPOLATION_MAJORITY 按多数值重新取样像素。
INTERPOLATION_NEARESTNEIGHBOR 按最近邻法重新取样像素。
构造方法详细信息

esri.layers.ImageServiceParameters()

创建新的ImageServiceParameters对象。构造函数没有参数。
属性详细信息

<Number[]> bandIds

当前波段选择的数组。

<Number> compressionQuality

当前压缩质量值。压缩质量控制了有多少损失的图片。仅仅对JPG图片类型有效。

<Extent> extent

导出图像的范围。

<String> format

地图图片格式。
已知值: png8 | png24 | jpg | bmp | gif
默认值: png8

<Number> height

请求图像高（单位：像素）。

<String> interpolation

当前插值法。插值法影响当栅格数据集变形或改变坐标空间时，栅格数据集如何变换。

<Number> width

请求的图像的高（单位：像素）。
