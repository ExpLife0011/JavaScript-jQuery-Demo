类: SpatialReference

描述

地图，图层或任务输入和输出的空间参考。投影和地理坐标系统都可以通过well-known ID (WKID)或definition string (WKT)定义。 JavaScript API 仅仅支持WKID。对于支持的ID列表和ID相关的定义字符串，参见：
  • 投影坐标系统列表
  • 地理坐标系统列表
