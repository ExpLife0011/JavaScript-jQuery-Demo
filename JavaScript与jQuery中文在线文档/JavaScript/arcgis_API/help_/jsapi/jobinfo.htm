类: JobInfo

描述

表示与服务器上异步GP任务的执行有关的信息。JobInfo没有构造方法。

类层次结构

esri.tasks.JobInfo

属性表

属性 类型 描述
jobId String ArcGIS Server分配的唯一工作ID。
jobStatus String 工作状态。返回值见常量表。
messages GpMessage[] 信息的数组，包含信息类型和描述。

常量

常量 描述
STATUS_CANCELLED 工作被取消。
STATUS_CANCELLING 工作正在取消中。
STATUS_DELETED 工作被删除。
STATUS_DELETING 工作正在删除中。
STATUS_EXECUTING 工作正在被工作处理器执行中。
STATUS_FAILED 工作执行失败。
STATUS_NEW 工作是新的。
STATUS_SUBMITTED 工作被提交。
STATUS_SUCCEEDED 工作成功完成。
STATUS_TIMED_OUT 工作执行超时。
STATUS_WAITING 工作正等待可用工作处理器。
属性详细信息

<String> jobId

ArcGIS Server分配的唯一工作ID。

<String> jobStatus

工作状态。返回值见常量表。

<GpMessage[]> messages

信息的数组，包含信息类型和描述。
