类: IdentifyTask

描述

通过ArcGIS Server REST API提供的地图服务资源的图层执行一个identity操作。

类层次结构

esri.tasks.IdentifyTask

构造函数

构造函数 描述
esri.tasks.IdentifyTask(url) 创建一个新的IdentifyTask对象. URL是一个必需的参数。

属性表

属性 类型 描述
url String 指向ArcGIS Server REST地图服务资源的URL。使用Services Directory获得URL。

方法表

方法 返回值 描述
execute(identifyParameters, callback?, errback?) dojo.Deferred 发送一个请求到ArcGIS REST地图服务资源来基于IdentifyParameters指定的参数执行identify要素的操作。完成时，onComplete时间被触发并且可选的callback函数被调用。

事件

事件 描述
onComplete(identifyResults) identify操作完成时触发该事件。
onError(error) 当执行任务发生错误时触发该事件。
构造方法详细信息

esri.tasks.IdentifyTask(url)

创建一个新的IdentifyTask对象. URL是一个必需的参数。
参数:
<String> url 必需的 指向ArcGIS Server REST地图服务资源的URL。 一个示例是 http://sampleserver1.arcgisonline.com/ArcGIS/rest/services/Portland/Portland_ESRI_LandBase_AGO/MapServer. 关于构造URL的更多信息，见 服务目录和REST API.
属性详细信息

<String> url

指向ArcGIS Server REST地图服务资源的URL。使用Services Directory获得URL。
方法详细信息

execute(identifyParameters, callback?, errback?)

发送一个请求到ArcGIS REST地图服务资源来基于IdentifyParameters指定的参数执行identify要素的操作。完成时，onComplete时间被触发并且可选的callback函数被调用。

dojo.Deferred返回结果在v1.4增加。
返回值: dojo.Deferred
输入参数:
<IdentifyParameters> identifyParameters 必需的 指定用于identify要素的条件。
<Function> callback 可选项 方法完成后调用的函数。 函数里的参数和onComplete事件里的参数一致。
<Function> errback 可选项 如果在任务执行期间，如果服务器端发生错误，error对象被返回。 (As of v1.3)
参见:
onComplete   onError
