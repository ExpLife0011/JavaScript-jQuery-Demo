类: BufferParameters

描述

为缓存操作设置距离，单位和其他参数。

类层次结构

esri.tasks.BufferParameters

构造函数

构造函数 描述
esri.tasks.BufferParameters() 创建一个新的BufferParameters对象。 构造函数没有参数。

属性表

属性 类型 描述
bufferSpatialReference SpatialReference 几何体被缓存所使用的空间参考。支持的空间参考列表，见投影坐标系统和地理坐标系统。

如果bufferSpatialReference没有指定，几何体用outSpatialReference指定的空间参考被缓存。如果outSpatialReference也没有指定，则使用features的空间参考。
distances Number[] 输入features被缓存的距离。距离单位通过unit指定。
features Graphic[] 输入图形或要素将缓存。
outSpatialReference SpatialReference 返回几何体的空间参考。 支持的空间参考列表，见投影坐标系统和地理坐标系统。

如果outSpatialReference没有指定，输出几何体使用bufferSR指定的空间参考，如果bufferSpatialReference也没有指定，使用features的空间参考。
unionResults Boolean 如果是true，所有在给定距离的缓存几何体被合并到单个(possibly multipart)多边形中，并且合并的几何体被放置在输出数组中。
unit Number 计算每个缓存距离的单位。如果unit没有指定，由bufferSpatialReference得到单位，如果bufferSpatialReference没有指定，由features的空间参考得到单位。单位值见常量表。
